13.至少说六点interface和abstract的区别。
  前者没有构造器 后者有,但是不能实例化
  前者所有方法都是public 后者可以是其他修饰
  前者允许多继承 后者不允许
  前者很少有成员变量,有也是常量 后者一般都有
  前者没有私有成员变量 后者可以有
  
  static 方法是类方法,它是不允许子类覆写（override）
而abstract方法是没有实现的方法，是必须由子类来覆写的。
